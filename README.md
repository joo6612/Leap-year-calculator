Leap Year Calculator
A simple program to determine whether a given year is a leap year.

Features
Checks if the input year is a leap year or not.

Handles input validation.

Supports a wide range of years.

How It Works
A year is a leap year if it is:

Divisible by 4,

Except years divisible by 100,

Unless they are divisible by 400.

For example, 2000 and 2024 are leap years, but 1900 and 2023 are not.

Usage
Clone the repository or download the code.

Run the script.

Enter the year when prompted.

See the result indicating if the year is a leap year.
